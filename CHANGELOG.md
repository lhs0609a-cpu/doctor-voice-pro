이 파일은 프로젝트의 모든 주요 변경사항을 기록합니다.

## [2.0.0] - 2025-10-30

### 추가됨 (Added)
- 🚀 프로덕션급 자동 배포 시스템
- 🛡️ 자가 진단 및 자동 복구 시스템
- 🚨 고급 에러 핸들링 및 리포트 생성
- 📦 원클릭 설치 스크립트 (Windows/Linux/Mac)
- 📊 성능 모니터링 시스템 (CPU, 메모리)
- 💾 자동 백업 시스템
- 📝 로그 로테이션 시스템
- 🔍 포괄적인 로깅 시스템 (Frontend + Backend)

### 개선됨 (Improved)
- ✨ AI 제안 기능 (Claude AI 통합)
- 🎨 SEO 분석 및 점수 시스템
- 🏷️ 태그 관리 시스템
- ⭐ 즐겨찾기 기능
- 🔍 고급 검색 및 필터링
- 🌓 다크 모드 지원
- 💾 자동 저장 기능
- ⚡ 스켈레톤 로딩 UI
- ⌨️ 키보드 단축키 시스템

### 기술 스택 (Tech Stack)
- Backend: FastAPI + SQLAlchemy + SQLite
- Frontend: Next.js 14 + React + TypeScript
- AI: Anthropic Claude API
- UI: Radix UI + Tailwind CSS
