# 네이버 블로그 자동 포스팅 완벽 가이드

## 🎯 개요

AI가 생성한 의료 콘텐츠를 **이미지와 함께** 네이버 블로그에 바로 복붙할 수 있도록 최적화된 시스템입니다.

### 주요 기능

✅ **형광펜 + 볼드 강조** (네이버 블로그 100% 호환)
✅ **이미지 자동 인식** (jpg, png, gif, webp)
✅ **이미지 자동 배치** (AI가 문단 사이에 적절히 삽입)
✅ **워드 문서 출력** (복붙만 하면 완성)
✅ **저장된 글 관리** (생성한 글을 나중에 사용)

---

## 📋 사용 흐름

```
[1단계] AI 글 생성
    ↓
[2단계] 저장된 글 탭에서 확인
    ↓
[3단계] 이미지 폴더 업로드
    ↓
[4단계] 워드 문서 다운로드
    ↓
[5단계] 워드 열어서 전체 복사
    ↓
[6단계] 네이버 블로그에 붙여넣기 - 완성!
```

---

## 1단계: AI 글 생성

### 글 작성 페이지에서

1. **대시보드 → 글 작성** 메뉴 클릭
2. 원본 의료 정보 입력
3. 각색 레벨, 프레임워크 등 설정
4. **"생성하기"** 버튼 클릭
5. AI가 블로그용 콘텐츠 생성

### 자동 기능

- 중요 키워드 자동 추출
- 의료법 위반 단어 검사
- SEO 최적화 (DIA/CRANK)
- 설득력 점수 분석

---

## 2단계: 저장된 글 관리

### 자동 저장

생성된 글은 자동으로 **로컬 스토리지에 저장**됩니다.

### 저장된 글 탭

**대시보드 → 저장된 글** 메뉴에서:

- 모든 생성된 글 목록 확인
- 날짜별 정렬
- 클릭해서 선택
- 필요없는 글 삭제

---

## 3단계: 이미지 폴더 업로드 ⭐

### 지원 형식

- JPG, JPEG
- PNG
- GIF (움짤 포함)
- WEBP

### 업로드 방법

1. **저장된 글 탭**에서 글 선택
2. **"이미지" 탭** 클릭
3. **"이미지 폴더 업로드"** 버튼 클릭
4. 이미지가 들어있는 폴더 선택
5. 모든 이미지 파일 자동 인식

### 이미지 배치 전략

#### 옵션 1: 문단 사이 배치 (추천 ⭐)

```
문단 1
문단 2
[이미지 1]
문단 3
문단 4
문단 5
[이미지 2]
문단 6
```

- 2-3 문단마다 이미지 삽입
- 자연스러운 흐름
- 가독성 최고

#### 옵션 2: 균등 분포

```
[이미지 1]
텍스트 33%
[이미지 2]
텍스트 33%
[이미지 3]
텍스트 33%
```

- 전체 길이를 이미지 개수+1로 나눔
- 균형잡힌 배치

### 이미지 캡션 자동 생성

파일명에서 자동으로 캡션 생성:

```
dental_clinic_01.jpg → "dental clinic"
임플란트_시술_전.png → "임플란트 시술 전"
photo_123.gif → "photo"
```

---

## 4단계: 워드 문서 다운로드

### 다운로드 버튼

1. **"다운로드" 탭** 클릭
2. **"네이버 블로그용 워드 다운로드"** 버튼 클릭
3. `.docx` 파일 자동 다운로드

### 파일명

```
{글제목}_with_images.docx
예: 치아건강관리법_with_images.docx
```

---

## 5단계: 워드에서 복사

### 워드 파일 열기

1. 다운로드된 `.docx` 파일 더블클릭
2. Microsoft Word 또는 한글 오피스에서 열림

### 전체 복사

```
Windows: Ctrl + A (전체 선택) → Ctrl + C (복사)
Mac: Cmd + A (전체 선택) → Cmd + C (복사)
```

### 확인 사항

열린 워드 문서에서 다음을 확인하세요:

✅ 키워드가 **노란 형광펜 + 볼드**로 강조
✅ 강조 문구가 **초록 형광펜 + 볼드**
✅ 인용구가 **회색 박스**로 표시
✅ 제목이 **크고 굵게**
✅ 이미지가 **문단 사이에 적절히 배치**
✅ 이미지 캡션이 **작은 회색 글씨**로 표시

---

## 6단계: 네이버 블로그에 붙여넣기

### 네이버 블로그 열기

1. https://blog.naver.com 접속
2. 로그인
3. **"글쓰기"** 버튼 클릭

### 붙여넣기

1. 에디터가 **기본 모드**인지 확인 (HTML 모드 X)
2. 커서를 본문 영역에 클릭
3. **Ctrl + V (Windows)** 또는 **Cmd + V (Mac)**
4. 짜잔! 완성!

### 결과 확인

✅ 형광펜 강조 → 그대로 유지
✅ 볼드 → 그대로 유지
✅ 인용구 → 회색 박스 유지
✅ 이미지 → 모두 삽입됨
✅ 이미지 위치 → 문단 사이에 적절히 배치
✅ 캡션 → 이미지 아래 작은 글씨로 표시

## 💡 팁 & 트릭

### 팁 1: 이미지 파일명 짓기

**좋은 예:**
```
치과_진료실.jpg
임플란트_시술_전.png
스케일링_과정_01.gif
```

**나쁜 예:**
```
IMG_1234.jpg  → 캡션이 "IMG"로 표시됨
ㄱㄴㄷㄹ.png  → 캡션 인식 불가
```

### 팁 2: 이미지 순서

파일명 순서대로 배치됩니다:

```
01_before.jpg
02_during.jpg
03_after.jpg
```

숫자를 앞에 붙이면 순서 제어 가능!

### 팁 3: 이미지 개수

**권장 비율:**
- 1000자당 이미지 1~2개
- 2000자 글 → 이미지 2~4개 추천

너무 많으면 부담스럽고, 너무 적으면 밋밋합니다.

### 팁 4: 글 재사용

한 번 생성한 글은 **저장된 글 탭**에 계속 남아있어서:

- 다른 이미지 조합으로 재생성 가능
- 수정 후 다시 저장 가능
- 여러 버전 관리 가능

## 🔧 문제 해결

### Q1: 워드에서 복사했는데 형광펜이 안 나와요

**A:** 다음을 확인하세요
1. 워드 파일을 제대로 열었는지
2. **전체 선택** (Ctrl+A) 후 복사했는지
3. 네이버 에디터가 **기본 모드**인지 (HTML 모드 X)
4. 브라우저: Chrome 또는 Edge 사용 권장

### Q2: 이미지가 업로드 안 돼요

**A:** 확인사항
- 파일 형식: JPG, PNG, GIF, WEBP만 가능
- 파일 크기: 개당 10MB 이하 권장
- 폴더 선택: 여러 파일 선택 필요

### Q3: 이미지 순서가 이상해요

**A:** 파일명 앞에 숫자 붙이기
```
01_first.jpg
02_second.jpg
03_third.jpg
```

### Q4: 캡션이 영어로 나와요

**A:** 파일명을 한글로 변경
```
dental_clinic.jpg → 치과_클리닉.jpg
```

### Q5: 형광펜 색상을 바꾸고 싶어요

**A:** 네이버 블로그에 붙여넣은 후:
1. 형광펜 부분 선택
2. 에디터 상단의 **"형광펜"** 버튼 클릭
3. 원하는 색상 선택
