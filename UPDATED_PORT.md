## 🛠️ 포트 다시 변경하려면

### 다른 포트로 변경
```bash
# 서버 종료 (Ctrl+C)

# 원하는 포트로 실행
cd doctor-voice-pro/frontend
npm run dev -- -p 3002  # 3002번 포트
npm run dev -- -p 3003  # 3003번 포트
```

### 기본 포트(3000)로 되돌리기
```bash
cd doctor-voice-pro/frontend
npm run dev
```
