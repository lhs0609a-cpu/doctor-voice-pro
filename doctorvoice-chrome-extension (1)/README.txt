📌 이 프로그램은 뭔가요?
─────────────────────────
저장된 글을 네이버 블로그에 자동으로 올려주는 크롬 확장 프로그램입니다.
클릭 한 번으로 제목, 본문, 이미지, 인용구, 배경색까지 자동 적용!


🚀 설치 방법 (아주 쉬워요!)
─────────────────────────

【1단계】 크롬 확장 프로그램 페이지 열기

   크롬 주소창에 아래 주소를 복사해서 붙여넣기 하세요:

   👉 chrome://extensions

   또는 크롬 메뉴(⋮) → 도구 더보기 → 확장 프로그램


【2단계】 개발자 모드 켜기

   오른쪽 위에 "개발자 모드" 스위치를 켜주세요 (파란색으로 바뀜)


【3단계】 확장 프로그램 추가하기

   방법 A: ZIP 파일인 경우
   ───────────────────────
   1. ZIP 파일 압축 풀기
   2. "압축해제된 확장 프로그램을 로드합니다" 클릭
   3. 압축 푼 폴더 선택

   방법 B: 폴더인 경우
   ───────────────────
   1. "압축해제된 확장 프로그램을 로드합니다" 클릭
   2. 이 폴더 선택


【4단계】 설치 완료! 🎉

   크롬 주소창 오른쪽에 보라색 아이콘이 나타나면 성공!
   (안 보이면 퍼즐 모양 🧩 클릭 → 핀 고정)


📝 사용 방법
─────────────

【준비】네이버 블로그 로그인
   먼저 네이버 블로그에 로그인 해주세요
   https://blog.naver.com


【1단계】닥터보이스 프로에서 글 선택

   1. 닥터보이스 프로 사이트 접속
   2. "저장된 글" 탭 클릭
   3. 포스팅하고 싶은 글 찾기
   4. "블로그 포스팅" 버튼 클릭


【2단계】확장 프로그램에서 설정

   1. 크롬 오른쪽 위의 보라색 아이콘 클릭
   2. 선택한 글 정보가 표시됨
   3. 옵션 설정:
      ✓ 인용구 자동 삽입 - 중요 문장에 인용구 스타일
      ✓ 중요 부분 강조 - 배경색으로 하이라이트
      ✓ 이미지 자동 삽입 - 본문 중간에 이미지 배치
      ✓ 임시저장 - 바로 발행하지 않고 임시저장


【3단계】포스팅 시작!

   "네이버 블로그에 포스팅하기" 버튼 클릭
   → 자동으로 네이버 블로그 열림
   → 제목, 본문, 서식 자동 입력
   → 완료!


⚠️ 주의사항
─────────────

• 네이버 블로그에 미리 로그인 되어 있어야 합니다
• 처음 사용할 때 잠시 시간이 걸릴 수 있어요
• 자동 입력 후 내용을 한번 확인해주세요
• 이미지는 URL이 있는 경우에만 자동 삽입됩니다


❓ 문제가 생겼나요?
─────────────────

Q: 확장 프로그램이 안 보여요
A: 크롬 퍼즐 아이콘(🧩) 클릭 → 닥터보이스 프로 핀 고정

Q: "글 정보가 없습니다"라고 나와요
A: 닥터보이스 프로에서 "블로그 포스팅" 버튼을 먼저 클릭하세요

Q: 네이버 블로그에 입력이 안 돼요
A: 네이버 로그인 확인 → 페이지 새로고침 → 다시 시도

Q: 에디터 형식이 이상해요
A: 네이버 블로그 에디터 버전에 따라 다를 수 있습니다
   수동으로 서식 조정이 필요할 수 있어요
