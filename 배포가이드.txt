【 방법 1: 완전 자동 설치 (가장 쉬움! ⭐) 】

  ✅ Python도, Node.js도 자동으로 설치됩니다!
  ✅ 프로그래밍 지식 필요 없음!
  ✅ 클릭 몇 번이면 끝!

  1️⃣ ZIP 파일을 다른 컴퓨터로 복사
     (USB, 이메일, 클라우드 등 아무거나 사용)

  2️⃣ 압축 풀기
     원하는 위치에 압축 해제

  3️⃣ "완전자동설치.bat" 실행
     - 파일 우클릭
     - "관리자 권한으로 실행" 선택
     - 자동으로 모든 것이 설치됩니다 (5-10분 소요)

  4️⃣ 끝!
     자동으로 브라우저가 열립니다.


【 방법 2: 부분 자동 설치 】

  ⚠️ Python/Node.js가 이미 설치되어 있는 경우

  1️⃣ ZIP 파일 압축 해제
  2️⃣ install.bat 더블클릭
  3️⃣ run.bat 더블클릭


【 방법 3: Python/Node.js만 자동 설치 】

  ⚠️ Python/Node.js만 없는 경우

  1️⃣ "auto_install_prerequisites.bat" 우클릭
  2️⃣ "관리자 권한으로 실행" 선택
  3️⃣ 컴퓨터 재시작
  4️⃣ install.bat → run.bat 실행

  ✅ Node.js 20.11.1 LTS (최신 장기 지원 버전)
  ✅ 필요한 모든 패키지
  ✅ 데이터베이스 초기화
  ✅ 서버 자동 실행

모든 것이 자동으로 설치됩니다!
