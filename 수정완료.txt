문제점:
  1. 특수 문자(╔, ║, ✅, ⚠️ 등)가 명령어로 인식됨
  2. install.bat 파일 경로를 찾지 못함

해결 내용:
  1. 모든 특수 문자를 [OK], [ERROR], [WARNING] 등으로 변경
  2. 파일 경로를 절대 경로(%~dp0)로 수정
  3. 배치 파일이 어디서 실행되든 올바른 위치를 찾도록 개선

================================================================

업데이트된 파일:
  - 완전자동설치.bat (수정됨)
  - auto_install_prerequisites.bat (수정됨)
  - DoctorVoicePro_Portable_20251031_053847.zip (새로 생성)

================================================================

테스트 방법:
  1. 새로 생성된 ZIP 파일 압축 해제
  2. 압축 해제한 폴더에서 "완전자동설치.bat" 우클릭
  3. "관리자 권한으로 실행" 선택
  4. 정상 작동 확인

================================================================

변경 사항:

이전:
  echo ╔════════════════╗
  echo ✅ Python 설치 확인 완료

이후:
  echo ================================================================
  echo [OK] Python 설치 확인 완료

================================================================

경로 수정:

이전:
  call install.bat

이후:
  call "%~dp0install.bat"

(%~dp0 = 배치 파일이 있는 디렉토리 경로)
