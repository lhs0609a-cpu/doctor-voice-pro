## 최종실행파일5의 특별한 기능

최종실행파일5는 "로그인 없이 바로 사용"할 수 있는 버전입니다.

핵심 차이점:
✓ 로그인 페이지 없음
✓ 회원가입 불필요
✓ http://localhost:3001/ 접속 시 자동으로 /dashboard/create로 이동
✓ 글작성 페이지에 바로 접근
✓ 인증 없이도 모든 기능 사용 가능

결과: 설치 후 바로 글작성 시작!

=====================================================================

## 자동으로 수행되는 작업

최종실행파일5.bat 실행 시:

[1/6] 포트 검색
  - 빈 포트 자동 탐색
  - 백엔드/프론트엔드 포트 할당

[2/6] 포트 정리
  - 사용 중인 포트 종료
  - 충돌 방지

[3/6] 로그인 없이 사용 설정
  ★ 백엔드: 인증 체크 제거
  ★ 프론트엔드: 메인 페이지 → /dashboard/create 리다이렉트
  ★ 프론트엔드: Dashboard 로그인 체크 제거

[4/6] 환경 변수 업데이트
  - .env 파일 자동 생성
  - 포트 설정 자동 적용

[5/6] 서버 시작
  - 백엔드 시작 (0.0.0.0:8010)
  - 프론트엔드 시작 (포트 3001)

[6/6] 연결 확인
  - 백엔드 연결 체크
  - 프론트엔드 연결 체크
  - 브라우저 자동 실행

## 백업 및 복원

### 자동 백업

최종실행파일5 실행 시 자동으로 백업됩니다:

백엔드:
- backend/app/api/deps.py.backup

프론트엔드:
- frontend/src/app/page.tsx.backup
- frontend/src/app/dashboard/layout.tsx.backup

### 인증 복원

로그인 기능을 다시 활성화하려면:

"인증복원.bat" 실행

또는 수동 실행:
python setup_no_auth.py --restore

복원 후:
- 다시 로그인 페이지가 나타납니다
- 회원가입/로그인이 필요합니다
- 기존 최종실행파일1~4 사용 가능

## 문제 해결

문제: 글작성 페이지가 안 나타남
해결:
  1. 포트정리.bat 실행
  2. 최종실행파일5.bat 다시 실행
  3. 브라우저 캐시 삭제

문제: 인증 오류 발생
해결:
  1. 인증복원.bat 실행
  2. 다시 최종실행파일5.bat 실행

문제: 로그인 페이지로 리다이렉트됨
해결:
  - 인증 복원이 되어 있습니다
  - 인증 없이 사용하려면:
    최종실행파일5.bat 다시 실행

문제: 백업 파일 복원하고 싶음
해결:
  1. 인증복원.bat 실행
  또는
  2. 수동 복원:
     backend/app/api/deps.py.backup → deps.py
     frontend/src/app/page.tsx.backup → page.tsx
     frontend/src/app/dashboard/layout.tsx.backup → layout.tsx
